SQL SELECT & Filtering Exercise – Summary

This project is a beginner-level SQL exercise focused on building foundational data querying skills using the employees table. The assignment introduced how to extract, filter, and organise data using core SQL clauses that are essential for any data analyst or database practitioner.

Purpose of the Assignment

The main goal of this exercise was to develop a strong understanding of how to retrieve meaningful information from a relational database using SQL. It focused on learning how to structure queries correctly and interpret expected outputs before executing them.

Key Skills Covered
Through this exercise, the following SQL concepts were practiced:

Using SELECT to retrieve all columns or specific fields from a table
Removing duplicate values using DISTINCT
Sorting results using ORDER BY in both ascending and descending order
Limiting results using LIMIT to control output size
Filtering data using WHERE conditions
Combining multiple conditions using AND, OR, NOT, and IN
Predicting query results before execution to strengthen analytical thinking

Dataset Used
All questions were based on a single dataset: the employees table, which contained information such as:

Employee ID
First and last names
Department
Salary
Hire date
City

Task Structure
The assignment consisted of 10 structured questions divided into two sections:

Selecting & Sorting Data, focusing on retrieving, ordering, and limiting results
Filtering with WHERE Clause, focusing on narrowing down results using conditions

For each question, the learner was required to:

Write the SQL query using correct syntax and formatting
Manually predict and draw the expected output table
Ensure correct column names and structured results
 
Overall Learning Outcome
By completing this exercise, I strengthened my ability to:

Read and interpret SQL queries
Understand how filtering and sorting affect data output
Structure clean and correct SQL statements
Think like a data analyst by predicting results before execution

This foundational exercise helped me to build essential skills required for more advanced data analysis, reporting, and database querying tasks.
